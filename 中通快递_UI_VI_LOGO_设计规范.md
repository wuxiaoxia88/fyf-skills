# 中通快递品牌 UI / VI / LOGO 配色设计规范
版本：V1.0  
适用范围：中通快递相关数字化产品、运营看板、海报物料、门店宣传、培训资料、轻量品牌延展  
文档属性：内部设计规范建议稿

---

## 1. 文档说明

本文件用于统一中通快递相关项目中的品牌视觉表达，覆盖以下三类场景：

1. **LOGO 使用规范**
2. **VI 视觉识别规范**
3. **UI 数字产品设计规范**

本规范基于中通公开品牌视觉特征整理，重点服务于以下实际业务场景：

- 网点经营看板
- 客服与签收数据系统
- 小程序 / H5 / 后台系统
- 培训海报与会议物料
- 网点门头、横幅、易拉宝、名片、工牌
- 自媒体内容封面与行业宣传页

> 说明：本文件属于内部落地型设计规范，用于统一实际项目输出。若后续获得中通官方完整品牌手册，应以官方版本为最高优先级。

## 3. 品牌配色系统

## 3.1 核心原则

中通品牌视觉的核心应围绕 **品牌蓝 + 白色 + 深色文字** 建立。  
整体风格应偏向 **清晰、效率、可信赖、现代物流科技感**。

---

## 3.2 官方识别基准色

以下颜色用于作为品牌基准色，优先级最高：

| 色彩角色 | 色值 | RGB | 用途 |
|---|---:|---:|---|
| 品牌主蓝 Primary Blue | `#2989FF` | 41, 137, 255 | LOGO 主色、主按钮、主标题强调、品牌背景 |
| 品牌白 White | `#FFFFFF` | 255, 255, 255 | LOGO 反白、页面底色、文字反白 |
| 深色文字 Dark Text | `#1F2937` | 31, 41, 55 | 正文、标题、后台系统主要文字 |
| 辅助灰 Border Gray | `#DCE6F2` | 220, 230, 242 | 边框、分割线、输入框描边 |
| 浅底蓝 Light Blue | `#EAF4FF` | 234, 244, 255 | 卡片底色、标签底色、状态底板 |

---

## 3.3 推荐扩展品牌色

以下为适配数字产品和运营物料的建议扩展色，可作为内部统一标准使用：

| 色彩角色 | 色值 | 用途 |
|---|---:|---|
| 品牌深蓝 | `#1565D8` | Hover、深色标题、重点模块底色 |
| 品牌浅蓝 | `#7DB7FF` | 图表辅助色、图形渐变第二色 |
| 品牌极浅蓝 | `#F4F9FF` | 页面大面积背景、数据区域底色 |
| 强调橙 | `#FF9F1A` | 时效提醒、待处理预警、运营高亮 |
| 警示红 | `#F04438` | 异常、超时、投诉、风险告警 |
| 成功绿 | `#12B76A` | 达成、已签收、完成、通过 |
| 信息青 | `#06AED4` | 补充状态、提示信息、次要科技模块 |

---

## 3.4 色彩使用比例建议

| 颜色类型 | 建议占比 | 说明 |
|---|---:|---|
| 品牌蓝系 | 20% | 负责品牌识别与核心动作 |
| 白色 / 浅底 | 60% | 保证页面清爽、可读 |
| 中性灰黑 | 15% | 负责内容表达与结构层级 |
| 功能状态色 | 5% | 仅用于状态反馈，避免滥用 |

---

## 3.5 渐变建议

中通品牌视觉可有限使用蓝色渐变，用于首页首屏、活动页横幅、招商海报主视觉。

**推荐渐变：**

```css
linear-gradient(135deg, #1565D8 0%, #2989FF 55%, #7DB7FF 100%)
```

**适用场景：**

- 活动主 KV
- 登录页首屏
- 大屏数据头图
- 会议横幅背景
- 招商宣传页

**不适用场景：**

- LOGO 主体
- 正文背景大面积通铺
- 表格单元格背景
- 信息密集后台主操作区

## 4. LOGO 使用规范

## 4.1 LOGO 版本

建议至少建立以下 4 套标准版本：

| 版本 | 说明 | 场景 |
|---|---|---|
| 标准彩色版 | 蓝色主体 + 白色文字 | 白底、浅底、宣传物料 |
| 反白版 | 白色 LOGO | 深蓝底、深色照片底 |
| 单色深色版 | 深灰 / 黑色单色 | 黑白打印、雕刻、单色工艺 |
| 图标简化版 | 仅圆形图标或核心字母图形 | App 图标、头像、角标、favicon |

---

## 4.2 安全留白

以 LOGO 圆形图标直径或字标高度的 **1/4** 作为最小安全留白。  
安全区内不得出现以下元素：

- 文字
- 图标
- 分割线
- 照片主体
- 复杂纹理

---

## 4.3 最小使用尺寸

| 类型 | 最小尺寸建议 |
|---|---|
| 屏幕端完整 LOGO | 宽度不小于 96px |
| 屏幕端图标版 | 宽度不小于 24px |
| 印刷端完整 LOGO | 宽度不小于 20mm |
| 印刷端图标版 | 宽度不小于 8mm |

---

## 4.4 背景适配规则

| 背景类型 | 推荐 LOGO |
|---|---|
| 白底 / 浅灰底 | 标准彩色版 |
| 品牌蓝底 / 深色底 | 反白版 |
| 照片背景 | 优先加纯色底板后放置 LOGO |
| 复杂纹理底 | 不建议直接使用 |

---

## 4.5 禁止事项

以下做法应明确禁止：

- 拉伸变形
- 随意改变比例
- 给 LOGO 添加阴影、描边、浮雕
- 修改蓝色为非品牌色
- 在复杂花纹上直接叠加
- 旋转 LOGO
- 使用低清晰度截图版 LOGO
- 将 LOGO 与其他字样硬拼接成新图形

## 5. VI 视觉识别规范

## 5.1 字体系统

### 中文字体建议

| 类型 | 推荐字体 | 用途 |
|---|---|---|
| 标题字体 | 思源黑体 / 阿里巴巴普惠体 / 苹方 | 海报标题、模块标题、数据标题 |
| 正文字体 | 思源黑体 / 苹方 / 微软雅黑 | 正文、说明、表格内容 |
| 数据字体 | DIN / Inter / Roboto | 数字、金额、KPI、大屏数据 |

### 英文字体建议

| 类型 | 推荐字体 | 用途 |
|---|---|---|
| 英文标题 | Inter / Montserrat | 国际化页面、标题 |
| 英文正文 | Inter / Roboto | UI 正文、表格、说明 |
| 数字专用 | DIN / Inter | 数据看板、统计数值 |

### 字重建议

| 层级 | 字重 |
|---|---|
| 主标题 | 700 |
| 二级标题 | 600 |
| 正文 | 400 / 500 |
| 数据数字 | 600 / 700 |

---

## 5.2 字号层级建议

| 层级 | PC端 | 移动端 |
|---|---:|---:|
| 一级标题 | 32-40 | 24-28 |
| 二级标题 | 24-28 | 20-22 |
| 三级标题 | 18-20 | 16-18 |
| 正文 | 14-16 | 14-16 |
| 说明文字 | 12-13 | 12-13 |
| KPI 大数字 | 36-64 | 28-40 |

---

## 5.3 圆角与线条

| 元素 | 建议值 |
|---|---|
| 大卡片圆角 | 16px |
| 常规卡片圆角 | 12px |
| 按钮圆角 | 10px |
| 输入框圆角 | 8px |

风格建议偏 **稳重、规整、现代**。  
不建议使用过度拟物风格和夸张装饰风格。

## 5.4 图标风格

| 维度 | 规范 |
|---|---|
| 风格 | 线性 + 少量面性结合 |

| 统一性 | 同一系统禁止混用多套风格 |

适配主题建议：

- 寄件
- 运单
- 运输
- 仓储
- 时效
- 异常
- 签收
- 客服
- 账单
- 数据看板
- 预警
- 网点管理

---

## 5.5 图片与插画风格

### 图片风格建议

- 真实物流场景
- 干净整洁的操作场地
- 快递员、客服、分拣、运输、仓配协同
- 强调效率、标准、专业感
- 避免杂乱背景和过度情绪化画面

### 插画风格建议

- 扁平化科技风
- 轻 3D 物流科技风
- 蓝白主色体系
- 强结构感、少装饰

## 6. UI 设计规范

## 6.1 UI 设计原则

| 原则 | 说明 |
|---|---|
| 先清晰后装饰 | 物流系统以信息可读性为先 |
| 先效率后炫技 | 操作效率高于视觉花样 |
| 先状态后美观 | 用户首先要看懂异常、时效、完成度 |
| 先统一后扩展 | 先建立组件库，再做专题活动页 |

---

## 6.2 页面布局建议

### PC 后台布局

| 区域 | 建议 |
|---|---|
| 顶部导航 | 品牌 LOGO + 一级菜单 + 用户信息 |

| 内容区 | 卡片化模块 + 表格 + 图表 |
| 右侧区域 | 辅助提醒、预警、快捷入口 |

### 移动端布局

| 区域 | 建议 |

| 顶部 | 页面标题 + 返回 + 操作按钮 |
| 主体 | 纵向卡片流 |
| 底部 | 关键导航 3 到 5 个以内 |
| 强调信息 | 使用状态标签和数据数字优先展示 |

## 6.5 状态色映射建议

| 业务状态 | 颜色 | 说明 |
|---|---|---|
| 已签收 / 已完成 | 绿色 | 正向完成态 |

## 9. 快速执行版规范

若团队时间有限，可直接先执行以下 10 条：

1. LOGO 统一只用标准版和反白版。
2. 全部页面主色统一为 `#2989FF`。
3. 页面底色统一优先白色或极浅蓝。
4. 标题统一用深色，正文避免纯黑。
5. 所有按钮统一一套圆角和状态。
6. 表格表头统一浅蓝底。
7. 异常统一红色，待跟进统一橙色，完成统一绿色。
8. 图表优先蓝色系，避免多色混乱。
9. 海报与 PPT 保持蓝白主调，减少杂色。
10. 网点宣传、系统界面、自媒体封面全部使用同一品牌色体系。

## 11. 一句话总结

这套规范的核心只有一句话：

**以中通品牌蓝为识别核心，以白色与深色文字建立清晰层级，以状态色承载物流业务信息，以统一组件提升系统效率与品牌一致性。**
