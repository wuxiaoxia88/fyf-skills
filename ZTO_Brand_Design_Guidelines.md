## 1. 品牌概述

### 1.1 品牌简介

中通快递股份有限公司（ZTO Express Co., Ltd.）成立于 2002 年 5 月 8 日，总部位于上海青浦，是一家集快递、物流及综合服务于一体的大型集团公司，注册商标为「中通®」与「ZTO®」。中通快递已在纽约证券交易所上市（NYSE: ZTO），业务规模稳居国内快递行业前列。

### 1.2 品牌使命与愿景

| 项目 | 内容 |
|------|------|
| **品牌使命** | 为客户提供高效、可靠、便捷的快递物流服务 |
| **品牌愿景** | 成为全球一流的智慧物流服务商 |
| **品牌口号（中文）** | 誉满中华，通达天下 |
| **品牌口号（国际）** | Anywhere（中通国际） |
| **核心价值观** | 诚信 · 高效 · 创新 · 共赢 |

### 1.3 品牌个性

中通快递的品牌个性可归纳为以下五个维度：

- **专业可靠** — 稳定的服务品质，值得信赖的物流伙伴
- **高效快捷** — 速度是核心竞争力，行动即结果
- **国际视野** — 立足中国，服务全球，具备战略胸怀
- **科技驱动** — 以互联网技术与大数据赋能物流全链路
- **亲和温暖** — 关注每一个包裹背后的人与情感

## 2. LOGO 标志规范

### 2.1 标志设计理念

中通快递标志将品牌创意完整融入「ZTO」三个字母之中，整体造型简洁大方，具备高辨识度与强视觉冲击力。

**设计内涵解读：**

- **Z** — 取自「中通」拼音首字母，同时也是英文字母表中最后一个字母，象征物流链条从 A 到 Z 的最后一公里服务
- **T** — 代表「中通（ZhongTong）」，体现企业的中国属性与本土根基
- **O** — 象征循环与全球，代表完整的物流产业链闭环，以及面向全球的开放格局
- **整体倾斜处理** — 字母组合采用倾斜角度排列，凸显速度感与方向感，直观传达快递行业「快捷」的行业属性

### 2.2 标志结构

```
┌─────────────────────────────────────────┐
│                                         │
│   [ZTO图形标志]   中通快递              │
│                  ZTO EXPRESS            │
│                                         │
└─────────────────────────────────────────┘
```

标志由以下三部分构成：

1. **图形标志（Symbol）** — 经过图形化处理的「ZTO」字母组合，带有倾斜动态感
2. **中文标准字（Logotype CN）** — 「中通快递」定制字体，笔画简洁有力
3. **英文标准字（Logotype EN）** — 「ZTO EXPRESS」辅助英文识别

### 2.3 标志比例与安全区

**标准横向组合：**

```
←——————————— 全宽 (W) ———————————→
┌──────────────────────────────────┐
│  [图形] │ 中通快递               │  高度 = W × 0.22
│         │ ZTO EXPRESS            │
└──────────────────────────────────┘
     ↑
  保护区 = W × 0.1（四周）
```

**最小使用尺寸：**

| 媒介类型 | 最小宽度 |
|----------|----------|
| 印刷品 | 20mm |
| 数字屏幕 | 60px |
| 车身标识 | 200mm |
| 名片 | 15mm |

### 2.4 标志使用版本

| 版本名称 | 适用场景 | 背景要求 |
|----------|----------|----------|
| 标准全彩版 | 白色/浅色背景 | 白色或浅灰 |
| 深色反白版 | 深色背景、夜间场景 | 深蓝、黑色 |
| 单色黑版 | 黑白印刷、刻章 | 任意 |
| 单色白版 | 烫印、丝印 | 深色 |

## 5. VI 视觉识别系统

### 5.1 基础设计元素

#### 图形元素

- **主图形**：ZTO 字母组合图形，可作为底纹、水印使用
- **线条元素**：细斜线/流线型线条，象征快捷与速度
- **几何辅助形**：圆角矩形（圆角半径 = 高度 × 0.08），用于标签、徽章

#### 渐变规范

**主渐变（品牌蓝渐变）：**

```
从左到右：#003580 → #1A5FA8 → #5B9BD5
角度：135°
用途：背景大图、Banner、主视觉
```

**辅助渐变（蓝绿渐变）：**

```
从左到右：#1A5FA8 → #72BF44
角度：120°
用途：特殊活动、节日宣传物料
```

### 5.2 名片设计规范

**标准名片尺寸**：90mm × 54mm

**正面布局：**
- 右上角：ZTO 标志（宽度约 30mm）
- 左下区域：姓名（H3）、职位（Body-M，次文字色）
- 底部：联系方式（Body-S）

**背面布局：**
- 居中：ZTO 标志全彩版或深底反白版
- 品牌主色填充背景（`#1A5FA8`）

### 5.3 工作服装规范

| 服装类型 | 主色 | 辅色 | Logo位置 |
|----------|------|------|----------|
| 快递员制服 | ZTO 蓝（#1A5FA8） | 白色 | 胸口左侧、后背居中 |
| 夏季T恤 | 白色 | ZTO 蓝条纹 | 胸口左侧 |
| 安全背心 | ZTO 橙黄（反光） | ZTO 蓝 | 背面居中 |
| 管理人员 | 深蓝（#003580） | 白色 | 胸口左侧 |

### 5.4 运输车辆涂装规范

**标准厢式货车：**

```
车身颜色分区：
┌──────────────────────────────────────────┐
│  上半部：ZTO蓝 #1A5FA8（占车身65%）      │
│                                          │
│         [ZTO 大型 LOGO 居中]             │
```

### 5.5 快递包装规范

## 6. UI 界面设计规范

### 6.1 设计原则

中通快递数字产品（App / 小程序 / Web）遵循以下设计原则：

| 原则 | 说明 |
|------|------|
| **清晰高效** | 信息层级分明，核心操作路径≤3步 |
| **品牌一致** | 色彩、字体、图标与品牌规范保持统一 |
| **移动优先** | 优先保障移动端体验，再向桌面端延伸 |
| **包容可及** | 支持无障碍访问，色彩对比度满足 WCAG AA 标准 |

## 8. 禁止使用规范

### 8.1 LOGO 禁止行为

以下行为严格禁止：

| 禁止项 | 说明 |
|--------|------|
| ❌ 拉伸变形 | 不允许改变 LOGO 横纵比例 |
| ❌ 旋转倾斜 | 不允许额外旋转或倾斜标志 |
| ❌ 添加阴影 | 不允许为 LOGO 添加投影或光晕效果 |
| ❌ 更改颜色 | 不允许使用非品牌规范色 |
| ❌ 添加描边 | 不允许为 LOGO 单独添加外描边 |
| ❌ 组合其他 LOGO | 不允许将 ZTO 与其他品牌 LOGO 并列组合 |
| ❌ 置于复杂背景 | 不允许直接放置于干扰可读性的花哨背景上 |
| ❌ 局部截取 | 不允许只使用 LOGO 的部分元素 |
| ❌ 像素化展示 | 不允许以低分辨率、模糊状态展示 LOGO |

### 8.2 色彩禁止行为

- ❌ 不得以任何渐变方式改变 ZTO 蓝的色相
- ❌ 品牌主色不得与黄色、大红等强反差色直接搭配
- ❌ 不得使用荧光色、过饱和色代替品牌色
- ❌ 深色背景上不得使用深色版 LOGO

### 8.3 字体禁止行为

- ❌ 不得以宋体、楷体等衬线字体作为品牌主字体
- ❌ 不得在品牌物料中使用仿宋、隶书等传统字体
- ❌ 不得使用版权不明的免费字体
- ❌ 标准字（LOGO 字体）不得单独提取用于其他排版
