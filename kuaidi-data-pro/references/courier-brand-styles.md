# 快递企业品牌风格库

> 8大中国快递企业的数据报告视觉风格。基于各企业官方VI品牌色规范整理。
> 当用户提到企业名称（如"顺丰风格"、"中通风格"）或"快递/物流品牌风格"时使用对应风格。

## 使用方式

1. 用户指定企业名 → 使用对应企业风格
2. 用户说"快递/物流风格"但未指定企业 → 从8种中推荐最适合的
3. 用户未提及快递/物流 → 继续使用通用5种风格（`report-style-gallery.md`）

选择后在回复开头说明，如「本次使用顺丰速运品牌风格」。

所有风格均适用于 `1200×675pt`（16:9报告页）或 `1200×900px`（信息图）的 Playwright 截图场景。

## 风格 1：中通快递 ZTO Express（双蓝科技）

> 一句话：冷色调双蓝体系，用深浅蓝色层次传递科技感和专业信赖。

### 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#F0F6FC` | 极浅蓝灰底，科技感底色 |
| 卡片/内容区 | `#FFFFFF` | 纯白卡片 |
| Header | `#0055B8` | 中通深蓝（PANTONE 285 PC） |
| 主要文字 | `#1A1A2E` | 深蓝黑 |
| 次要文字 | `#6B7B8D` | 蓝灰 |
| 辅色 | `#5DC8F0` | 浅天蓝（PANTONE 2925 PC），品牌标签/渐变条 |
| 强调蓝 | `#1A87DE` | 明亮蓝，数据高亮 |
| 正向 | `#0055B8` | 用主色表示正向 |
| 负面 | `#CC3333` | 柔和红 |
| 边框 | `#D0DFF0` | 浅蓝边框 |

### 字体

全部使用 `'Helvetica Neue', Arial, 'PingFang SC', sans-serif`

### 关键设计元素

- **蓝色渐变条**：Header下方4px渐变条（`#0055B8` → `#5DC8F0` → `#1A87DE`），中通最标志性的视觉元素
- **蓝色顶线KPI卡片**：每个KPI卡片有3px蓝色顶线，交替使用深蓝/浅蓝/明亮蓝
- **蓝色左边框洞察卡片**：4px `#0055B8` 左边框 + 白色背景
- **深蓝行动卡片**：`#0055B8` 背景 + `#5DC8F0` 标题
- **品牌标签**：浅天蓝底 + 深蓝字的圆角标签
- **表头深蓝底白字**，斑马纹 `#F5F9FF`

### ECharts色板


## 风格 5：极兔速递 J&T Express（红白极简）

> 一句话：红白双色极简留白，国际化干净感，强调速度与技术实力。

### 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#FFFFFF` | 纯白底 |
| 浅灰区域 | `#F5F5F5` | 区块分隔 |
| Header | `#E60012` | 极兔红（PANTONE 485 C） |
| 主要文字 | `#1A1A1A` | 深黑 |
| 次要文字 | `#777777` | 中灰 |
| 深红层次 | `#CC0000` | 深红，层次对比 |
| 正向 | `#E60012` | 用主红表示增长 |
| 负面 | `#999999` | 灰色弱化负面 |
| 边框 | `#EEEEEE` | 极浅灰边框 |

### 字体

全部使用 `'Helvetica Neue', Arial, 'PingFang SC', sans-serif`

### 关键设计元素

- **红色Header + 大量留白**：简洁Header，不过度装饰
- **红白极简KPI**：白底无边框，红色大数字 + 极细灰色标签
- **极简表格**：无边框，仅底部细线分隔，红色高亮关键值
- **红色左边框引用块**：3px `#E60012` 左边框 + 白色背景
- **最大化留白**：内边距50px+，段距24px+
- **无阴影、无圆角、无渐变**：极简到极致

### SF Express 与 J&T Express 区分

两家均用红色为主色，但风格截然不同：

| 维度 | 顺丰 SF | 极兔 J&T |
|------|---------|---------|
| 背景 | 黑色Header，白色内容 | 红色Header，白色内容 |
| 红色角色 | 红色在黑底上做标题 | 红色做大色块Header |
| 辅助色 | 黑色、深灰 | 浅灰、极浅灰 |
| 整体感觉 | 高端沉稳、权力感 | 国际极简、速度感 |
| 装饰度 | 中等（洞察卡片、Exhibit编号） | 极低（大量留白） |
